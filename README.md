# feasible-set-collapse-theory
## Motivation

（为什么要讨论这个问题）

在许多现实决策系统中，人们面临的失败并非源于错误选择，而是因为可行选择本身已经消失。

现实中大量决策失败呈现出一个共同特征：

参与者并非不理性

也并非信息不足

却在长期上遭遇系统性失败

主流解释通常诉诸：

激励不足

认知偏差

执行力问题

但这些解释默认一个前提：

“仍然存在一个可被理性选择的行动空间”

本文关注一种被系统性忽视的情形：
并非行动者选择不当，而是可供选择的理性行动集合已经塌缩。

本研究并不试图提出更优策略，而是试图回答一个更基础的问题：
在什么条件下，继续参与本身已经不再是理性的？

## What Is a Feasible Set?

（什么叫“可行域”）

这里的目标：

把“可行域”说成一个任何理工/社科读者都能接受的抽象对象。

可写展开：

在决策问题中，可行域指的是：

在给定约束下

能够被实际执行

且不会立即导致系统性失败的行动集合

可行域不同于：

偏好集合（preferences）

最优解集合（optima）

一个行动可以是：

理论上最优

但在现实约束下不可行

在本文中，“可行”并不意味着最优，而仅意味着：
该行动在当前约束条件下仍然允许系统存续。

关键区分（很重要）：

可行 ≠ 高收益

可行 ≠ 成功

可行 = 不被结构性淘汰

## When Do Feasible Options Disappear?

（可行选项什么时候会消失）

3.1 Constraint Accumulation（约束累积）

规则、成本、时间压力、竞争强度不断叠加

单个约束尚可应对，组合约束导致可行域骤减

3.2 Irreversibility（不可逆性）

一次性窗口（考试、周期、资格）

失败后无法回到初始状态

导致“重试”不再是一个选项

3.3 Endogenous Feedback（内生反馈）

参与者行为本身改变系统结构

如：拥挤、内卷、过度竞争

最终使原本可行的策略失效

可行域塌缩往往不是线性的，而是呈现出阈值特征：
在某一临界点后，多数理性行动同时失效。


## Survival vs. Optimization

（存活逻辑 vs 优化逻辑）

核心论点只有一句：

在可行域正在塌缩的系统中，存活优先于优化。

多数决策理论关注：

如何提高效率

如何接近最优

这些方法隐含假设：

系统将继续存在

决策者拥有调整空间

而 FSCT 关注的是：

当系统面临结构性淘汰风险时，
决策的首要问题不再是“做得多好”，
而是“是否还能继续”。

在此意义上，优化失败并非理论缺陷，
而是其适用前提已经不再成立。

## What This Framework Does NOT Do

（这套框架不做什么）

本框架不提供：

精确预测

操作化策略

统一阈值或度量指标

本框架不声称：

所有系统都会塌缩

所有失败都是结构性的

个体努力没有意义

本文提出的是一种判别视角，
而非一套可执行算法。

## Open Questions

（开放问题）

可行域塌缩是否存在可观测的早期信号？

不同系统中，约束累积是否呈现相似结构？

可行域是否可能在外生冲击下重新扩张？

如何区分“暂时困难”与“结构性无解”？

本文并不试图回答上述问题，而是试图说明：
在继续优化之前，判断问题是否仍然值得被优化，本身就是一个独立且必要的研究层级。
